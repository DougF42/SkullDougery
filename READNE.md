
  This project looks at input from the camera, and tracks any
  motion. The expectation is that the camera is FIXED, any
  changes (motion) is of interest.
 
  lucas_sparse.cpp is the original example file
 
  tracker.cpp      adds my modifications to use the tracking
                   info to control SkullDougery. 
  Configuration.cpp/Configuration.h define a class where various
                   program paramters can be hard coded.
